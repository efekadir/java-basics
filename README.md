# Java Fundamentals – Practice Projects

This repository contains a collection of beginner-friendly Java console applications created while learning the fundamentals of Java programming. The projects focus on core programming concepts such as conditional statements, loops, methods, user input, mathematical calculations, and basic problem-solving.

The repository includes small standalone applications such as calculators, converters, simple games, and utility programs that helped reinforce fundamental Java concepts.

## 📂 Projects

| File                                   | Description                                                                                                           |
| :------------------------------------- | :-------------------------------------------------------------------------------------------------------------------- |
| `AngleCalculator.java`                 | Calculates the interior angles of a triangle using the coordinates of its three vertices.                             |
| `BMICalculator.java`                   | Calculates the Body Mass Index (BMI) from the user's height and weight, then displays the corresponding BMI category. |
| `ChineseZodiac.java`                   | Determines the user's Chinese Zodiac sign based on the entered birth year.                                            |
| `ComputeCircleArea.java`               | Calculates the area of a circle from its radius and validates against negative input values.                          |
| `CurrentTimeCalculator.java`           | Uses `System.currentTimeMillis()` to convert elapsed milliseconds into years, days, hours, minutes, and seconds.      |
| `HexToDeciAdvanced.java`               | Converts a single hexadecimal digit entered by the user into its decimal equivalent.                                  |
| `LoanPaymentCalculator.java`           | Calculates monthly loan payments based on the loan amount, annual interest rate, and loan term.                       |
| `Lottery.java`                         | A simple lottery game where the user attempts to match a randomly generated two-digit number.                         |
| `OneDigitSum.java`                     | Generates two random single-digit numbers and repeatedly prompts the user until the correct sum is entered.           |
| `PerpendicularBisectorCalculator.java` | Calculates the equation of the perpendicular bisector of a line segment using the coordinates of two points.          |
